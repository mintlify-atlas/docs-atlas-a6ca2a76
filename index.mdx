## Chart Categories

### Web Services
Deploy general-purpose web applications with nginx, specialized platforms like helpdesk and archery, and modern AI applications with dify.

### Monitoring & Exporters
Export metrics from cloud providers (Aliyun, Tencent Cloud, UCloud, Qiniu, Upyun) and services (Beanstalkd, Memcached, Tinyproxy) to Prometheus.

### Infrastructure
Essential Kubernetes components including CoreDNS, Gatekeeper policy controller, CSI drivers, and load balancer controllers.

### Applications
Ready-to-deploy applications including Kafka UI, Selenium grid, NoneBot chatbot, and various development tools.

## Quick Example

<CodeGroup>

```bash Add Repository
helm repo add douban https://douban.github.io/charts/
helm repo update
```

```bash Search Charts
helm search repo douban
```

```bash Install a Chart
helm install my-nginx douban/nginx
```

```bash Install with Custom Values
helm install my-app douban/nginx \
  --set image.tag=1.25.0 \
  --set ingress.enabled=true \
  --set ingress.hosts[0].host=example.com
```

</CodeGroup>
